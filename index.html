📊 Proyecto Churn de Clientes

Has sido contratado como asistente de análisis de datos en Telecom X. La empresa enfrenta una alta tasa de cancelaciones y necesita comprender los factores que llevan a la pérdida de clientes.

El objetivo es recopilar, procesar y analizar datos para encontrar patrones que expliquen por qué los clientes abandonan el servicio.

⚙️ Proceso ETL

Extracción

Importación de datos desde una API utilizando Python.

Transformación

Limpieza de datos, tratamiento de valores nulos y normalización.

Carga

Preparación del dataset para análisis exploratorio y modelos predictivos.

📈 Análisis Exploratorio de Datos

  • Identificación de patrones de cancelación
  • Análisis de variables demográficas
  • Relación entre servicios contratados y churn
  • Segmentación de clientes
